# Tool CLI

A custom Node.js CLI tool built for learning, experimentation, and small projects.  
This repo contains:

- **tool/** → The CLI tool itself  
- **testProject/** → A sample project demonstrating usage of the CLI

---

## Features

- `--start` → Start the app
- `--info` → Display information about the app
- Debug logging support using `DEBUG=*`

---

## Prerequisites

- Node.js v18+ recommended
- npm installed

---

## Installation

### Option 1: Global install (recommended)

Install the CLI globally to use `tool` from anywhere:
